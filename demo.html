Web Games

// 1. Install
npm install @substream/sdk

// 2. Initialize (demo mode)
import { Substream } from '@substream/sdk'
await Substream.init('demo')

// 3. Start streaming
const live = await Substream.startLive({
  source: { type: 'canvas', canvas: gameCanvas }
})

// 4. Stop streaming
await live.stop()

Features

  • WebRTC ultra-low latency (<1s)
  • Automatic quality adaptation
  • Works with any Canvas/WebGL game
  • No server setup required
  • Cross-platform viewers

Unity/Quest Games

// Drop the SDK prefab into your scene
// Then in your code:

await Substream.Init(new SubstreamConfig {
    BaseUrl = "demo"
});

var live = await Substream.LiveCreate(new LiveOptions {
    Width = 1920, Height = 1080, Fps = 60
});

await live.Start();
